# TinyNote

A beginner-friendly Python project to learn basics of Python programming.

## Getting Started

### Prerequisites

- Python 3.6 or higher

### Installation

1. Clone or download this project
2. Navigate to the project directory
3. Run the application:

```bash
python3 main.py
```

## Usage

Once running, you can:
- **(l)ist** - View all your notes
- **(a)dd** - Create a new note
- **(v)iew** - Read a specific note
- **(d)elete** - Remove a note
- **(q)uit** - Exit the application

## Features

- Create and store notes
- List all notes
- View individual notes with timestamps
- Delete notes
- Notes are persisted to a JSON file

## Learning Objectives

- File I/O with JSON
- Functions and modularity
- Dictionaries and data structures
- User input handling
- Control flow (loops, conditionals)
